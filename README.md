# wps-office-langpack-ru

Пакет русификации для WPS Office 12 под Linux.

## Описание

В официальной версии WPS Office для Linux отсутствует русский язык интерфейса. Однако языковые файлы от Windows-версии полностью совместимы с Linux-версией.

Данный проект содержит исходные файлы и скрипт для упаковки оригинального языкового пакета и плагинов перевода WPS Office в формат `.deb`.

## Установка готового пакета

Готовую сборку можно установить из репозитория [apt.smbit.pro](https://apt.smbit.pro):

```bash
sudo apt install wps-office-langpack-ru
```

## Самостоятельная сборка

Для самостоятельной сборки `.deb`-пакета выполните:

```bash
./build.sh
```

Собранный `.deb`-файл будет помещен в директорию `dist/`.
